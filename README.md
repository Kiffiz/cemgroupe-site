# SITE INTERNET-CEM — CEMGROUPE

Site vitrine institutionnel de CEMGROUPE — Compagnie des Experts Maritimes du Groupe.

## Structure
- `index.html` — Page unique complète (HTML/CSS/JS intégré)

## Sections
1. Navigation fixe avec lien vers CEMPROG
2. Hero — Présentation groupe
3. Barre accréditations animée
4. À propos — Histoire & chiffres clés
5. Filiales — 6 agences (CI, BJ, TG, SN, CM, GH)
6. Services — 6 domaines d'expertise
7. Chiffres clés animés
8. Carte Leaflet interactive — Réseau portuaire
9. Direction Générale (Mme Bouraima + M. Fassinou)
10. Plateforme CEMPROG — présentation modules
11. Interventions — photos terrain
12. Témoignages partenaires
13. Formulaire de demande d'intervention
14. Footer complet

## Technologies
- HTML5 / CSS3 (variables CSS, Grid, Flexbox)
- JavaScript vanilla (IntersectionObserver, animations)
- Leaflet.js — Carte interactive
- Google Fonts — Playfair Display + DM Sans
- CartoDB Dark Matter — Tiles carte

## Design
- Palette : Navy #060F1F + Or #C9A227
- Typographie : Playfair Display (titres) + DM Sans (corps)
- Animations scroll reveal, compteur chiffres, zoom hero
- Responsive mobile (hamburger menu)
